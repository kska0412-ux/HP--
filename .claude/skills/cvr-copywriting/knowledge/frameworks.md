# コピーフレームワーク図鑑

14サイトの分析から抽出した、実戦投入可能なコピーフレームワーク全集。
各フレームワークに「定義 / 使用条件 / テンプレート / 失敗パターン / 実例典拠」を付す。

---

## 1. PASONA（パソナの法則）

### 定義
Problem（問題提起）→ Agitation（扇動・共感）→ Solution（解決策提示）→ Narrowing down（限定）→ Action（行動喚起）の5段構成。神田昌典氏が体系化した日本市場で最強の汎用フレーム。

### 使用条件
- ✅ 痛み・悩みが明確な商材（治療系・ダイエット・サロン・サプリ・スクール）
- ✅ 読者が課題自覚済み（Solution Aware〜Most Aware）
- ❌ 新カテゴリ商品（読者が問題を認識していない場合）
- ❌ 衝動買い商材・低単価ギフト

### テンプレート
```
【P】「{症状A}／{症状B}／{症状C} 」とお悩みのあなたへ
【A】{既存解への共感否定} / {放置すると起きる未来}
【S】そんなあなたに {ブランド名} の {独自手法} があります
【N】今月{限定◯名}様限定で、{特典・割引}をご用意
【A】{低関与CTA文言} ＞
```

### 失敗パターン
- 【A】の扇動が弱い：「お悩みですか？」だけで終わると共感トリガーが起動しない
- 【N】を入れ忘れる：限定性が抜けるとオファーの希少価値が下がる
- 【S】で根拠を出さず説明する：自慢に聞こえてしまう

### 実例典拠
- `anka-tokushima`：「どこへ行っても痛みが取れない…」(A)→「驚異の95%が実感」(S)
- `nobles`：「一人じゃ続かない…でも、諦めたくない！」(P+A)→「マンツーマン指導で理想のボディを手にいれる！」(S)→「まずは60分の無料体験」(A)
- `sango_kotsuban`：悩みタグ4種(P)→「痛みがない最善の施術で」(A打消し)→「8,000人超／97%」(S実証)→「LINEで無料チェック」(A)

## 2. AIDA（注意・関心・欲求・行動）

### 定義
Attention（注意）→ Interest（関心）→ Desire（欲求）→ Action（行動）の古典的フレーム。FVでの大数字・大型インパクトに最適。

### 使用条件
- ✅ コールド層・通りすがりトラフィック（広告・SNS流入）
- ✅ ビジュアル×ファクト系商材（治療実績・体験数・%系）
- ❌ 検討期間が長いBtoB商材（QUEST向き）

### テンプレート
```
【A】驚異の{インパクト数字}！
【I】{機能・特徴}が {結果} という新習慣／話題の{業界}{固有名} とは？
【D】{ベネフィット羅列3〜5点} / {未来時制で結果を約束}
【A】{無料体験／資料請求／LINE登録} ＞
```

### 失敗パターン
- 数字を文章中に埋没させる（必ずデカ文字で単独配置）
- Aで攻め過ぎてIの説明が薄くなる（「驚異の!」だけで終わる）

### 実例典拠
- `anka-tokushima`：「驚異の95%が実感」→「ガチガチ肩甲骨が1回で動く」→「徳島の施術知ってますか？」
- `seitai_sdk`：「楽30分でになる！」→「87,540人以上が体験した」
- `esthetic-sdk`：「わずか3ヶ月でトータル −42cm」→パーツ分解→オファー

## 3. QUEST（クエストの法則）

### 定義
Qualify（読者を絞る）→ Understand（理解共感）→ Educate（教育）→ Stimulate（刺激）→ Transition（移行・CTA）。Michael Fortin提唱のセールスコピー型。BtoB・高単価・専門商材向け。

### 使用条件
- ✅ 高単価／検討期間長／教育要素必須の商材（業務用機器、研修、スクール、コンサル）
- ✅ ターゲットが限定的でフィルタが効くと残った人が濃い
- ❌ 衝動買い系・低単価コスメ

### テンプレート
```
【Q】{制約条件}でも、{願望}を持つ {役割・属性} へ
【U】{反論1}も {反論2}も、もう前提ではありません
【E】{独自メカニズム/根拠}：なぜ{ブランド}が{結果}を実現できるのか
【S】{比喩}が手に入るとしたら？／月額{数字}で{価値}
【T】入力は{時間}、{障壁解除}：[資料請求][LINE]
```

### 失敗パターン
- Q（絞り込み）を省略 → CVR希釈
- E（教育）を長くしすぎる → 離脱
- Sを抽象的にする → 損失逃避が起動しない

### 実例典拠
- `emdual`：「資金に余裕がなくても、売れる新メニューを作りたいサロンオーナー様へ」(Q) →「ローン返済の不安も、スタッフ教育の悩みも、もう前提ではありません」(U) →「30分で結果が見える/放置中に別業務/解約も自由」(E) →「スタッフをもう一人雇ったような稼働感」(S) →「入力は1分、しつこい営業電話なし」(T)
- `revi-pro`：「メニュー導入で失敗したくないサロンオーナーへ」(Q) →「最小投資×最短スピード」(S) →「資料請求」(T)

## 4. PREP（結論先行型）

### 定義
Point（結論）→ Reason（理由）→ Example（事例）→ Point（再結論）。サプリ・栄養食品など、機能訴求に向く。

### 使用条件
- ✅ 機能・成分が明確で、結論を1行に圧縮できる商材
- ✅ 中盤〜末尾のセクション内ロジック（FV全体には弱い）
- ❌ 感情訴求が主軸の商材

### テンプレート
```
【P】{結論：これで{結果}}
【R】なぜなら {独自成分／メカニズム}
【E】事実：{第三者機関認定／実績数字／写真}
【P】だから {結論再強調} / 私らしく {ライフスタイル文}
```

### 実例典拠
- `koubo`：「酵母の鉄+亜鉛が腸からキク」(P)→「乳酸菌5,000万個配合／酵母由来」(R)→「厚生労働省の定めた栄養機能食品」(E)→「私らしく前向きに」(P)

## 5. ストーリー型（Hero's Journey圧縮版）

### 定義
Before（過去の主人公）→ Conflict（葛藤）→ Resolution（出会い・解決）→ Future（未来の状態）。読者を主人公に投影させる。

### 使用条件
- ✅ Before/Afterが視覚化できる商材（薄毛・痩身・スクール）
- ✅ 高関与・情緒コンバージョン商材
- ❌ 機能スペック比較が主軸のBtoB

### テンプレート
```
【B】"{主人公属性}の私"が、{挫折経験}
【C】{既存解への失望：美容室では聞けない／病院では様子を見ましょう／…}
【R】そんなとき出会ったのが {ブランド・手法}
【F】{時間}後、{変化}が始まります
```

### 失敗パターン
- 主人公属性が読者と乖離（年齢・性別・状況をプロットせずに書く）
- Conflictが弱いと感情の起伏が出ない

### 実例典拠
- `mito-hatsumou`：「"薬でも変わらなかった私"が、髪と心にふんわり 自信を取り戻した理由」(B+R)→「美容室では気をつかって聞けない／病院では"様子を見ましょう"で終わってしまう」(C)→「3ヶ月後、あなたの印象が変わり始めます」(F)
- `lucil-academy`：「もう、手に職をつけたいを諦めない」(B)→「好きを仕事に変える 新しい働き方を」(F)

## 6. カテゴリーキング型（既存カテゴリ否定→新カテゴリ提示）

### 定義
既存カテゴリ（競合）を「〜でもない」と三段否定し、自社を独自カテゴリとして位置づける。Al Ries & Jack Troutのポジショニング理論の応用。

### 使用条件
- ✅ レッドオーシャン市場で差別化が困難な商材
- ✅ コンセプトが独自で言語化できる手法
- ❌ コモディティ商品（USPが弱い場合は逆効果）

### テンプレート
```
{症状}、{"心"まで届いていませんか？}
{競合A}でも、{競合B}でも、{競合C}でもない
"{独自手法の象徴的動作}"なのに、{驚異の結果}
—— それが、{ブランド名・新カテゴリ名}
```

### 失敗パターン
- 否定する対象が読者の認知にない（マイナー競合だと知らない）
- 自社の独自性が言語化されておらず「結局◯◯系では？」と思われる

### 実例典拠
- `otona-stretch`：「整体でも、マッサージでも、運動でもない／"触れるだけ"なのに、呼吸まで深くなる。—— それが、大人の脱力ストレッチ」
- `theb`：「ただ痩せるだけじゃ、なんだか貧相になりそう」→「"整える"という新しい選択」

## 7. 二段CTA型（関与度分岐）

### 定義
高関与CTA（説明会・体験予約）と低関与CTA（資料DL・LINE登録）を併置し、関与度の異なるリードを両方拾う。

### 使用条件
- ✅ 検討期間が長い商材（資格スクール、業務用機器）
- ✅ リードナーチャリング前提のマーケティング設計
- ❌ 即時決済が主目的の単発キャンペーン

### テンプレート
```
[CTA高関与：オレンジ／緑] {時間}で疑問を解消 → 「{無料体験／説明会}を申し込む」
[CTA低関与：青／グレー] {時間}で完了 → 「{資料請求／LINE登録}」
```

### 実例典拠
- `gfa202602`：「40分で疑問を解消／無料で説明を聞いてみる」＋「簡単60秒で／無料で資料を受け取る」
- `emdual`：「フォームで問い合わせる」＋「LINEで問い合わせる」（チャネル分岐型）

## 8. 社会的証明物量型

### 定義
お客様の声写真／実績数字／メディア掲載／推薦人を**物量で**並べる。1つ1つは弱くても、視覚的密度で説得する。

### 使用条件
- ✅ 治療院・サロン・スクールなど信頼が購買決定要因の業種
- ✅ 第三者の写真が許諾済みで揃う事業者
- ❌ 法規制の厳しい医療領域（薬機法）

### テンプレート
```
[実績数字3点バッジ：累計実績／満足度／変化実感]
[メディアロゴ／推薦人ラベル：◯◯メダリスト推薦／◯◯誌掲載]
[お客様の声写真グリッド：24枚以上 / 顔出し+手書きコメント]
```

### 失敗パターン
- 写真の解像度がバラバラ／合成感
- 全員が女性／全員が男性など多様性を欠く
- "効果には個人差があります"の薬機注釈が抜ける

### 実例典拠
- `sango_kotsuban`：写真24枚以上＋「8,000人超／97%／98%」＋「東京五輪メダリストも推薦」
- `seitai_sdk`：「87,540人以上が体験」「実績20年」

## 9. オファー先行型（強オファー直撃）

### 定義
ベネフィット説明より先に、強烈なオファー（割引／限定数／返金保証）をFV直下に提示し、損失逃避を即起動。

### 使用条件
- ✅ 単発・体験型サービス（エステ初回／整体初回）
- ✅ ターゲットが価格主導で動く層
- ❌ 価値訴求が主軸の高関与BtoB

### テンプレート
```
{通常価格}円 → ★今だけ★ {割引率}OFF {体験価格}円
[★先着◯名様限定★]
[万が一のときも安心 全額返金保証付き]
```

### 実例典拠
- `mito-hatsumou`：「初回体験 3,980円／月5名限定」
- `otona-stretch`：「13,200円→今だけ 9,900円／先着30名／全額返金保証」
- `esthetic-sdk`：「通常18,000円→50%OFF→9,000円／毎月10名限定」

## 10. 制約条件付きYOUメッセージ型

### 定義
「[制約]でも、[願望]を持つ[役割]へ」の構造で、特定状況の読者だけを精密に呼び込む。

### 使用条件
- ✅ ニッチターゲット向けBtoB／高関与商材
- ✅ ターゲットの心理プロファイルが明確
- ❌ マスマーケット商材

### テンプレート
```
{制約条件：資金がなくても／時間がなくても／一人でも／未経験でも}、
{願望：売れる新メニューを作りたい／理想のボディを手にいれる／…}
{役割：サロンオーナー／30代女性／…} へ
```

### 実例典拠
- `emdual`：「資金に余裕がなくても、売れる新メニューを作りたいサロンオーナー様へ」
- `revi-pro`：「もう、メニュー導入で失敗したくないサロンオーナーへ」
- `nobles`：「一人じゃ続かない…でも、諦めたくない！とお悩みのあなたへ」

## フレームワーク選択チート

| 商材タイプ | 推奨フレーム | 補助フレーム |
|---|---|---|
| 治療院・整体 | PASONA | 大数字AIDA / 物量社会的証明 |
| 美容サロン体験 | オファー先行型 | PASONA / ストーリー型 |
| BtoB機器・サブスク | QUEST | 制約付きYOU / 二段CTA |
| 資格スクール | AIDA + 二段CTA | 制約付きYOU |
| ダイエット・痩身 | PASONA + 数字分解 | オファー先行型 |
| サプリ・健康食品 | PREP | 物量社会的証明 |
| 独自手法・新カテゴリ | カテゴリーキング型 | ストーリー型 |
| 産後・ママ向け | PASONA + 物量社会的証明 | オファー先行型 |
