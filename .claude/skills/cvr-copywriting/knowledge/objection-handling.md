# 購入障壁ハンドリング集

14サイトの分析から逆算した、購買意思決定を止める「障壁トップ10」と、それを潰すコピーの型集。

障壁ハンドリングは「読者が次に考える反論」を1秒先回りして潰し続ける作業。LP中盤〜FAQブロックで集中投下する。

---

## 障壁 1. 「高い」（価格抵抗）

### 障壁の心理メカニズム
読者は商品単体の価格ではなく、「自分の予算枠」「他選択肢の機会費用」と比較している。Kahneman のメンタルアカウンティング理論。値段そのものではなく、価値との比較感が問題。

### 潰しの型

**型A. アンカリング型**
```
通常価格 {高アンカー}円 → 今だけ {低価格}円（{割引率}%OFF）
{業界相場}円のところ、当社 {自社価格}円
```

**型B. 分割訴求型**
```
1日あたり {小単位}円
月額 {手の届く価格}円から
コーヒー1杯分の値段で
```

**型C. ROI訴求型（BtoB向き）**
```
{月額}円で、{回収シナリオ}が可能
{初期投資}が、{N}ヶ月で回収
スタッフをもう一人雇った時の人件費 vs 本サービス {月額}円
```

### 配置位置
- 価格セクションの直前または直後
- FAQブロックの最上段（「料金は高くないですか？」）

### 実例典拠
- `otona-stretch`：「13,200円→9,900円」（アンカリング）
- `esthetic-sdk`：「18,000円→9,000円（50%OFF）」（アンカリング＋割引明示）
- `emdual`：「スタッフをもう一人雇ったような稼働感」（ROI比較）
- `mito-hatsumou`：「初回体験 3,980円」（コーヒー数杯分のアンカー）

## 障壁 2. 「本当に効果あるの？」（効果懐疑）

### 障壁の心理メカニズム
過去の同種商品で失敗経験がある／業界全体への不信／薬機法商材で誇大広告に慣らされている。読者は「言い切り」を逆に怪しむ。

### 潰しの型

**型A. 数字証明型**
```
{累計人数}人が体験／{満足度}%が実感
{第三者機関}による{試験}で{結果数値}
{年数}年の実績
```

**型B. お客様の声物量型**
```
[顔出し写真24枚以上の声グリッド]
{属性}の{年代}の方：「{Before状態}が{After状態}に」
```

**型C. リスクリバーサル型**
```
万が一効果を実感できなければ、全額返金保証
{期間}間 無条件返金
```

### 配置位置
- FV直後〜中盤の信頼ブロック
- 体験申込CTA直前

### 実例典拠
- `sango_kotsuban`：「8,000人超／97%／98%」＋ 写真24枚＋ メダリスト推薦
- `seitai_sdk`：「87,540人以上が体験／実績20年」
- `otona-stretch`：「全額返金保証」
- `anka-tokushima`：「驚異の95%が実感」

## 障壁 3. 「自分には合わないかも」（属性ミスマッチ）

### 障壁の心理メカニズム
LPに登場する顧客像が自分と乖離していると「自分は対象外」と判断する。同調性が確保できないと購買決定が止まる。

### 潰しの型

**型A. ペルソナ呼びかけ型**
```
{年代}の{属性}でも、効果を実感しています
{制約条件}でも、{結果}が得られた例
こんな方が利用しています：
　・{属性パターン1}
　・{属性パターン2}
　・{属性パターン3}
```

**型B. 個別最適化訴求型**
```
あなた専用の{プログラム／メニュー}
マンツーマン／個別ヒアリングで設計
診断結果に基づくおすすめ
```

**型C. 多様性物量型**
```
[年代別・属性別のお客様の声を網羅的に並べる]
30代女性：…／40代男性：…／50代主婦：…／産後ママ：…
```

### 配置位置
- 共感ブロック（FV直後）
- お客様の声セクション

### 実例典拠
- `nobles`：「マンツーマン指導」（個別最適化）
- `emdual`：「資金に余裕がなくても」（制約あっても対象）
- `sango_kotsuban`：多様な産後ママの声写真群

## 障壁 4. 「続くか不安」（継続不安）

### 障壁の心理メカニズム
過去のジム・スクール・サブスクで挫折した経験がある層に強い。一人で頑張る不安・モチベ維持の困難さ。

### 潰しの型

**型A. 伴走訴求型**
```
マンツーマンで伴走
{頻度}のフォロー体制
専属トレーナー／コーチが付き添い
```

**型B. ハードル下げ型**
```
1回 {短時間}分から
週 {N}回でOK
スマホで完結／自宅で完結
```

**型C. 共同体訴求型**
```
仲間と一緒に頑張れる
コミュニティ機能で励まし合い
{N}人が同じ目標で進行中
```

### 配置位置
- 中盤の機能訴求ブロック
- FAQ「続けられるか不安です」

### 実例典拠
- `nobles`：「一人じゃ続かない…でも、諦めたくない！」→「マンツーマン指導」
- `seitai_sdk`：「楽になる30分」（時間ハードル下げ）
- `lucil-academy`：「新しい働き方」（仲間とのコミュニティ訴求）

## 障壁 5. 「今じゃなくていい」（先延ばし）

### 障壁の心理メカニズム
人は決断を先延ばしにする傾向（present bias）。緊急性がなければ後回しが快適。

### 潰しの型

**型A. 希少性訴求型**
```
★先着{N}名様限定★
毎月{N}名限定
{枠数}名様まで → 残り{N}枠
```

**型B. 緊急性訴求型**
```
今だけ {割引率}OFF
キャンペーン {終了日}まで
[カウントダウンタイマー]
```

**型C. 損失強調型**
```
{放置リスク：症状進行／機会損失／競合台頭}
今始めなければ、{時間}後にはさらに{悪化}
```

### 配置位置
- FV直下（即決層向け）
- 終盤CTA直前（最後の押し）

### 実例典拠
- `mito-hatsumou`：「初回体験 3,980円／月5名限定」
- `otona-stretch`：「13,200円→今だけ 9,900円／先着30名」
- `esthetic-sdk`：「毎月10名限定」

## 障壁 6. 「副作用・リスクが怖い」（安全性懐疑）

### 障壁の心理メカニズム
健康・医療・美容領域で特に強い。過去の副作用報道や口コミでネガティブ情報を持っている層。

### 潰しの型

**型A. 権威認証型**
```
{公的機関}が定めた{認定}
{医師／専門家}監修
{第三者試験}で安全性確認済み
```

**型B. 工程透明化型**
```
{原料}：{産地・由来}を公開
{製造工程}：国内{工場}で製造
{成分}：全成分表示
```

**型C. 既存解との比較型**
```
{侵襲的施術}とは違い、当社は{非侵襲}
{薬}とは違うアプローチ
副作用ゼロ／痛みなし／ダウンタイムなし
```

### 配置位置
- 製品紹介セクション
- FAQ「副作用はありますか？」

### 実例典拠
- `koubo`：「厚生労働省の定めた栄養機能食品」（公的認定）
- `sango_kotsuban`：「痛みがない最善の施術で」（非侵襲訴求）

## 障壁 7. 「時間がない」（時間負債）

### 障壁の心理メカニズム
共働き・子育て世代に強い。スケジュール管理の認知負荷を増やしたくない心理。

### 潰しの型

**型A. 短時間訴求型**
```
1回 {N}分から
{秒数}で完了の{アクション}
スキマ時間で／通勤中でも
```

**型B. オンライン完結型**
```
自宅で完結
スマホで予約・受講
来店不要／オンライン受講可
```

**型C. 並行可能訴求型**
```
{他作業}しながら{サービス利用}
ながら{機能}で時短
放置中に別業務
```

### 配置位置
- 機能訴求ブロック
- FAQ「忙しいですが続けられますか？」

### 実例典拠
- `seitai_sdk`：「楽になる30分」
- `emdual`：「30分で結果が見える／放置中に別業務」
- `gfa202602`：「簡単60秒で」「40分で疑問を解消」

## 障壁 8. 「場所が遠い／通えない」（物理的アクセス）

### 障壁の心理メカニズム
店舗型ビジネスでの典型障壁。通勤・通学経路と外れていると検討対象外になる。

### 潰しの型

**型A. オンライン代替型**
```
オンライン受講可
リモート診断
スマホで完結
```

**型B. 出張・宅配型**
```
出張対応 {エリア}
全国配送無料
{エリア}まで送料無料
```

**型C. 多拠点訴求型**
```
全国{N}店舗
最寄り駅から徒歩{N}分
{エリア}店もオープン
```

### 配置位置
- アクセス情報セクション
- FAQ「遠方でも利用できますか？」

### 実例典拠
- `lucil-academy`：オンライン受講可訴求
- `anka-tokushima`：「徳島」と地域限定訴求（ローカル特化）

## 障壁 9. 「解約・キャンセルが大変そう」（離脱不安）

### 障壁の心理メカニズム
過去のサブスク・継続課金で解約手続きに苦労した経験がある層。「捕まったら抜けられない」恐怖。

### 潰しの型

**型A. 解約自由訴求型**
```
いつでも解約自由
{期間}縛りなし
解約金不要
```

**型B. 手続き簡便訴求型**
```
解約は{N}クリックで完了
LINEから解約手続き可
電話不要／オンラインで完結
```

**型C. クーリングオフ訴求型**
```
{期間}間のクーリングオフ対応
{N}日以内の返品OK
```

### 配置位置
- 価格・契約セクション直下
- FAQ「解約方法は？」

### 実例典拠
- `emdual`：「解約も自由」
- `otona-stretch`：「全額返金保証」（実質的なクーリングオフ）

## 障壁 10. 「営業電話／勧誘が嫌」（接触リスク）

### 障壁の心理メカニズム
情報入力後にしつこい営業電話が来た経験がある層。個人情報を渡すこと自体への抵抗。

### 潰しの型

**型A. 明示的否定型**
```
※しつこい営業電話はありません
※強引な勧誘は一切しません
※当社からのご連絡は{メール／LINE}のみ
```

**型B. チャネル選択型**
```
LINEで完結／電話なし
お問い合わせは {メール / LINE / フォーム} から選択可
```

**型C. プライバシー訴求型**
```
個人情報は厳重管理
第三者提供なし
{プライバシーマーク}取得
```

### 配置位置
- フォーム入力欄直下
- CTAボタン直下の摩擦解消ラベル

### 実例典拠
- `emdual`：「しつこい営業電話なし」（CTA直下に明示）
- `gfa202602`：チャネル選択型の二段CTA

| 資格スクール | 続くか不安 / 効果懐疑 | 伴走訴求 / 数字証明 | 共同体訴求 |
| ダイエット・痩身 | 効果懐疑 / 続くか不安 | お客様の声物量 / 伴走訴求 | リスクリバーサル |

## 障壁ハンドリング配置マップ

LPの位置別に、どの障壁を潰すかの推奨配置。

```
[FV]
　主障壁：時間がない／場所が遠い
　→ 「{秒数}で」「オンライン完結」を冒頭から提示

[FV直下 共感ブロック]
　主障壁：属性ミスマッチ
　→ ペルソナ呼びかけ型／制約条件付きYOU

[機能訴求ブロック]
　主障壁：効果懐疑
　→ 数字証明型／第三者機関認定

[お客様の声ブロック]
　主障壁：効果懐疑／属性ミスマッチ
　→ 多様性物量／写真24枚以上

[料金セクション]
　主障壁：高い／解約不安
　→ アンカリング／分割訴求／解約自由

[FAQ]
　主障壁：上記すべて
　→ 各障壁を網羅的にカバー

[最終CTA]
　主障壁：先延ばし／営業電話
　→ 希少性／緊急性／しつこい営業なし
```
